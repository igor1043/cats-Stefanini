# Cats Stefanini
aplicação que consume e mostra imagens de gatos da api https://api.imgur.com

## Tecnologias utilizadas
- Linguagem Kotlin 
- Retrofit
- OkHTTP3
- Gson
- Coil
- LiveData
- Courotines Scopes
- MVVM
